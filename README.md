# VnHub

Visual Novel library manager for organizing and managing your visual novel collection.

## Features

-  **Library Management** - Organize and manage your visual novel collection
-  **Group Organization** - Create custom groups to categorize your VNs
-  **Database Storage** - Local SQLite database for persistent storage
-  **VNDB Integration** - Connect with VNDB for metadata and information
-  **Statistics** - View statistics about your collection
-  **Settings** - Customizable application settings
-  **Backup Service** - Backup and restore your library data
-  **Modern UI** - Web-based interface built with HTML/CSS/JavaScript

## Technology Stack

- **Language:** C# (.NET 8.0)
- **Framework:** Windows Desktop (WinForms/WPF with WebView2)
- **UI:** HTML5, CSS3, JavaScript
- **Database:** SQLite
- **Localization:** English, Japanese, Russian

## Getting Started

### Prerequisites

- .NET 8.0 SDK or Runtime
- Windows 10/11 or later

### Installation

1. Clone the repository:
```bash
git clone https://github.com/gqwg2003/Vn-Hub.git
cd Vn-Hub
```

2. Build the project:
```bash
dotnet build
```

3. Run the application:
```bash
dotnet run
```
